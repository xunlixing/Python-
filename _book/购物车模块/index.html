购物车模块 Cart

插入购物车

/**
*  userID：用户ID
*  productID：商品ID
*  designerID：设计师ID
*  return：插入结果
*/
Bool insertProduct(userID,productID,designerID){
    1. 如果用户没有购买此商品，直接插入商品，并设置数量为1
    2. 如果用户已购买过此商品，则直接增加数量
}
